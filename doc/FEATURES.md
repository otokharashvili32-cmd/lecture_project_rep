# Project Features & Functionalities

This document contains all possible features and functionalities for the project. This is for reference and planning purposes.

## Current Features

### Authentication & Navigation
- Welcome page with Log In and Sign Up buttons
- Email and password input forms
- "Maybe Later" option to skip authentication
- Main page navigation system (shows, discography, merch, about)

### Content Sections
- **Shows:** Displays random concert dates with cities
- **Discography:** Album display (AMAO) with cover image
- **Merch:** Product catalog with prices (T-Shirt $35, Vinyl $95, Guitar $1999)
- **About:** Band information and description

### Design & Styling
- Dark, cryptic, moody theme
- Tall, condensed typography (Oswald font)
- Glassmorphism effects
- Smooth animations and transitions
- Responsive layout

### Technical
- Backend: Node.js/Express server
- Frontend: React with Vite
- API proxy configuration
- Hardcoded data (no database yet)
